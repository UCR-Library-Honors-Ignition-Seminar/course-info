# UCR Library Honors Ignition Seminar

**Fabrication and Media Design: Making in the Physical and Digital World**

Welcome to the UCR Library Honors Ignition Seminar! This is a 10-week hands-on course where you will learn to make things — physical objects, games, interactive stories, AI-powered art — and build a public portfolio along the way.

## How This Course Works

Each week you will:
1. Attend two class sessions and complete in-class activities
2. Finish a **skill/technology-based project** related to that week's tools
3. Write a **300-word reflection** using the provided template
4. Upload both to your GitHub Classroom repository
5. Submit your GitHub repo link on **Canvas** for grading

**Final Project (Weeks 9-10):** Combine at least two skills from anywhere in the course into one project. Present it to the class with a short presentation or poster.

## Submitting Your Work

1. Open your GitHub Classroom repo (link on Canvas each week)
2. Add your project files and `reflection.md` to a `/week-XX/` folder
3. Commit your changes
4. Copy the repo URL and submit it on Canvas

> **New to GitHub?** Start with the [GitHub & GitHub Classroom Guide](GITHUB-GUIDE.md) — no coding experience needed.
